Métodos Estatísticos Multivariados

Repositório para versionamentos e armazenamentos das atividades propostas na disciplina LCE5859 - Métodos Esttísticos Multivariados, ministrada pelo professor Carlos Tadeu dos Santos Dias pelo Programa de Pós-Graduação em Estatística e Experimentação Agronômica da ESALQ-USP como curso de verão. Os exercícios são descritos no livro Métodos Estatísticos Multivariados: uma introdução [@Manly2008], que também é adotado como livro-texto na disciplina.
